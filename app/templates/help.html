This is a help page for Recallr

What is Recallr?

Recallr is a web application designed to help users make a decks and flashcards for studying. It allows users to create, manage, and review flashcards efficiently.

What does it do?

Recallr allows users to create decks of flashcards. Each deck has a name and a set of cards. Each card has a question and an answer. The platform also allows users to do quizzes to see how well they remember their flashcards. Results of quizzes can be reviewed by users to see the question they got wrong and the ones they got right.

How do you use it?

Below is a detailed guide to using every feature of Recallr.

add_circle

How do I create a deck?

  1. Go to the home page of Recallr.
  2. Click on get started.
  3. Click on the "Create Deck" button.
  4. Enter the name of the deck.
  5. Click on the "Create" button.
edit

How can I edit a deck?

  1. Go to the deck page.
  2. Click on the "Edit" button.
  3. Enter the new name for the deck.
  4. Click on the "Update" button.
delete_outline

How can I delete a deck?

  1. Go to the deck page.
  2. Click on the "Delete" button.
  3. This is going to open a confirmation modal.
  4. Click on the "Delete" button.

Note: This will delete all the cards in the deck.

add_box

How can I create a card?

  1. Go to the deck page.
  2. Click on the "Create Card" button.
  3. Enter the question for the card.
  4. Enter the answer for the card.
  5. Click on the "Create" button.
edit

How can I edit a card?

  1. Go to the deck page.
  2. Click on the "Edit Card" button.
  3. Enter the new question for the card.
  4. Enter the new answer for the card.
  5. Click on the "Update" button.
delete_outline

How can I delete a card?

  1. Go to the deck page.
  2. Click on the "Delete Card" button.
  3. This is going to open a confirmation modal.
  4. Click on the "Delete" button.

Note: This is a permanent action and cannot be undone.

How do I start a quiz?

  1. Go to the deck page.
  2. if the deck is empty add some cards to it.
  3. After you have added some cards, click on the "Start Quiz" button.
  4. After you have finished a quiz, you can review the results.

How do I review the results of a quiz?

  1. Go to the decks page.
  2. Click on the deck you want to review.
  3. Click on "Previous Results" button.
  4. You can then see a list of your previous quizzes.
  5. Click on the arrow at the end of quiz result row to see the quiz.
